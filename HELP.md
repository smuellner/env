## Git Shortcuts

| Command       | Description              |
| ------------- |--------------------------|
| `gc`          | git checkout             |
| `gco`         | git checkout origin      |
| `gcm`         | git checkout master      |
| `gpull`       | git pull                 |
| `gpush`       | git push                 |
| `gbranch`     | git checkout branch      |
| `gclone`      | git clone                |
| `gsub`        | git update all submodules|
| `gui`         | open the default git UI  |
